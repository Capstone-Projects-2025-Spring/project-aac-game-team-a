Class: StrokeData

StrokeData(xCoord, yCoord, color, size)

This class represents all data associated to an indiviudal stroke drawn on the canvas

Constructor

new StrokeData(xCoord, yCoord, color, size)

Creates an instance of a StrokeData object to store data of individual drawing strokes on the canvas
Parameters:
Name Type Description
xCoord number The X-coordinate of the drawing position on the canvas
yCoord number The Y-coordinate of the drawing position on the canvas
color string The selected color at the time of creation of StrokeData object
size number The size of the drawing used for proper scaling of the drawing
